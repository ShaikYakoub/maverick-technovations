## Recommended Specs

- Image format: JPG, quality 82-88
- Hero images: 2560x1440 minimum
- Video format: H.264 MP4, 24-30fps
- Loop videos: 6-10 seconds

### Hover Video Constraints

- All service-card videos must use: `preload="none"`, `muted`, `loop`, `playsInline`, with poster image.
- Desktop: activate on hover.
- Mobile/touch: activate only when card is centered in viewport; pause when leaving center zone.
